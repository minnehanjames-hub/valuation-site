# Valuation Workbench — LBO, DCF & Multiples Simulator

## In plain terms

An interactive, browser-only tool for learning how the three core valuation methods work — and how your assumptions change the answer. Drag sliders for things like leverage, growth rate, exit multiple, or discount rate, and instantly watch enterprise value, equity value, MOIC, and IRR update. It's a hands-on way to build intuition for finance and investment math.

## The three models

- **LBO (Leveraged Buyout)** — model returns (MOIC, IRR) on a debt-funded acquisition; tweak entry/exit multiples and leverage.
- **DCF (Discounted Cash Flow)** — value a business from projected cash flows, with PV of cash flows + terminal value and an EV→equity bridge.
- **Multiples** — value a company off an EV/EBITDA multiple.

## Run it

Just open `index.html` in a browser — no install or build step. Built with HTML, JavaScript, and Chart.js.

## Note

An educational model for building valuation intuition — not for real investment or diligence decisions.
